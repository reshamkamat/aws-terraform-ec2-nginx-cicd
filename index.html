AWS + Terraform + Nginx

Cloud Infrastructure Project with Terraform

This website is hosted on an AWS EC2 instance, provisioned using Terraform, and served through Nginx. The next phase of this project is CI/CD automation using GitHub Actions.

Project Highlights

AWS EC2

Deployed a cloud-based Linux server on AWS for hosting the web application.

Terraform

Provisioned infrastructure as code for repeatable, automated deployments.

Nginx

Configured Nginx as the web server to host and serve the application.

CI/CD Ready

Prepared this project for future deployment automation using GitHub Actions.

Built as part of a hands-on DevOps and Cloud Engineering learning journey.
